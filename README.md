# FitLife Gym - Fitness Client

Welcome to **FitLife Gym**, a modern and responsive fitness website built with **React**, **TailwindCSS**, and **DaisyUI**. This project is designed to provide a seamless user experience with a sticky navbar, responsive design, and a clean, professional interface.

🌐 **Live Demo**: [FitLife Gym](https://fitlifegym-theabh.netlify.app/)

---

## 🚀 Features

- **Sticky Navbar**: Always accessible navigation for a better user experience.
- **Responsive Design**: Fully optimized for all devices, including desktops, tablets, and mobile phones.
- **Hamburger Menu**: Mobile-friendly navigation with a collapsible menu.
- **Modern UI**: Styled with TailwindCSS and DaisyUI for a sleek and professional look.
- **Fast and Lightweight**: Built with Vite for blazing-fast performance.

---

## 🛠️ Tech Stack

- **Frontend**: React, TailwindCSS, DaisyUI
- **Build Tool**: Vite
- **Hosting**: Netlify
